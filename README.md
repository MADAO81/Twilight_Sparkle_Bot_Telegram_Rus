## 📖 О проекте

Сумеречная Искорка — это интерактивный бот, который:

- 📚 **Напоминает о делах** — главная фишка! Распознаёт даты и время в сообщениях
- 📖 **Советует книги** — книжный червь поделится любимыми произведениями
- 🌤️ **Показывает погоду** — всегда знает, что за окном
- 🔮 **Даёт магические советы** — немного волшебства для вдохновения
- ✨ **Желает спокойной ночи** — тёплое завершение дня
- 📚 **Присылает факт дня** — каждый день что-то новое и интересное
- 💬 **Общается с характером Искорки** — книжная, ответственная, слегка тревожная, но очень добрая

## 🚀 Быстрый старт

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/MADAO81/Twilight_Bot_Telegram_Rus.git
cd Twilight_Bot_Telegram_Rus
```

### 2. Создайте виртуальное окружение

```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate     # Windows
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Настройте переменные окружения

Скопируйте файл `.env.example` в `.env` и заполните его:

```env
# Telegram
TELEGRAM_TOKEN=ваш_токен_бота

# OpenAI
OPENAI_API_KEY=ваш_ключ
OPENAI_MODEL=gpt-4-turbo

# Координаты Ворсино
DEFAULT_LAT=55.0965
DEFAULT_LON=36.6355

# Настройки
WORK_START_HOUR=9
WORK_END_HOUR=22
CONTEXT_EXPIRE_DAYS=30

# Администратор
ADMIN_ID=ваш_telegram_id

# Режим отладки
DEBUG_MODE=false
```

### 5. Запустите бота

```bash
python run.py
```

## 📝 Особенности поведения

### 🎯 Реакция на сообщения в группах

| Ситуация | Поведение |
| :--- | :--- |
| Упоминание бота (`@username`) | ✅ Всегда отвечает |
| Ответ на сообщение бота | ✅ Всегда отвечает |
| Обычное сообщение (без упоминания) | ❌ Не отвечает |
| Личное сообщение | ✅ Всегда отвечает |

### ⏰ Рабочее время

* **Дни работы:** Понедельник — Воскресенье
* **Часы работы:** 9:00 — 22:00

> ⚠️ **Вне рабочего времени:** бот полностью игнорирует сообщения в группах. В личных сообщениях (ЛС) отвечает: *«Я отдыхаю, приходите завтра!»*
