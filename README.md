## 📖 О проекте

Сумеречная Искорка — это интерактивный бот, который:

- ⏰ **Напоминает о делах** — главная фишка! Поддерживает **личные** (в личку) и **групповые** (в группу) напоминания, а также **ежемесячные повторения** (например, «20-го числа каждого месяца»).
- 📖 **Советует книги** — книжный червь поделится любимыми произведениями.
- 🌤️ **Показывает погоду** — всегда знает, что за окном.
- 🔮 **Даёт магические советы** — немного волшебства для вдохновения.
- ✨ **Желает спокойной ночи** — тёплое завершение дня.
- 📚 **Присылает факт дня** — каждый день что-то новое и интересное.
- 💬 **Общается с характером Искорки** — книжная, ответственная, слегка тревожная, но очень добрая.

## 🧠 Напоминания: Как это работает

### Личные и групповые
- **Личное:** Бот отправит напоминание **только вам в личку**, даже если вы создали его в группе.  
  *Пример:* `@twilight_sparkle_rus_bot напомни сегодня в 19:00 про вечернюю тренировку`
- **Групповое:** Бот отправит напоминание **в группу**, где оно было создано.  
  *Пример:* `@twilight_sparkle_rus_bot групповое напоминание завтра в 10:00 провести встречу`

### Ежемесячные повторения
Искорка понимает фразу **«20-го числа каждого месяца»** и будет напоминать вам каждый месяц в указанный день и время.  
*Пример:* `@twilight_sparkle_rus_bot групповое напоминание 20-го числа каждого месяца в 9:00 передавать показания счётчиков`

### Умная отмена
Вы можете отменить напоминание по смыслу, даже если не помните точный текст.  
*Пример:* `@twilight_sparkle_rus_bot отмени напоминание про счётчики`

## 📝 Особенности поведения

### 🎯 Реакция на сообщения в группах

| Ситуация | Поведение |
|----------|-----------|
| Упоминание бота (`@username`) | ✅ Всегда отвечает |
| Ответ на сообщение бота | ✅ Всегда отвечает |
| Обычное сообщение (без упоминания) | ❌ Не отвечает |
| Личное сообщение | ✅ Всегда отвечает |

### ⏰ Рабочее время
**Понедельник — Воскресенье: 9:00 — 22:00**

Вне рабочего времени: игнорирует сообщения в группах, в личке отвечает: *«Я отдыхаю, приходите завтра!»*

## 🚀 Быстрый старт

### 1. Клонируйте репозиторий

```bash
git clone https://github.com
cd Twilight_Sparkle_Bot_Telegram_Rus
```

### 2. Создайте виртуальное окружение

```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate     # Windows
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Настройте переменные окружения

Скопируйте файл `.env.example` в `.env` и заполните параметры:

```env
# Telegram
TELEGRAM_TOKEN=ваш_токен_бота

# OpenAI
OPENAI_API_KEY=ваш_ключ
OPENAI_MODEL=gpt-4-turbo

# Координаты Ворсино
DEFAULT_LAT=55.0965
DEFAULT_LON=36.6355

# Настройки
WORK_START_HOUR=9
WORK_END_HOUR=22
CONTEXT_EXPIRE_DAYS=30

# Администратор
ADMIN_ID=ваш_telegram_id

# Режим отладки
DEBUG_MODE=false
```

### 5. Создайте базы данных

```bash
python3 bot/scripts/create_reminders_db.py
python3 bot/scripts/create_books_db.py
python3 bot/scripts/fill_books_db.py
```

### 6. Запустите бота

```bash
python run.py
```
